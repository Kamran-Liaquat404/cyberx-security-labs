## 🔐 CyberX

**CyberX** is a **frontend-based Cyber Security & Ethical Hacking educational platform** designed for **learning through simulations**.

The project follows an **80% education + 20% interactive practice** approach, allowing users to understand cybersecurity concepts in a **safe and legal environment**.

### 🛠️ Services & Labs:
- Ethical Hacking Linux Labs (commands & practice)
- Email Breach Awareness
- Port Scanner (Simulation)
- Cryptography Lab (Text Encrypt / Decrypt)
- IP Tracker (Educational)
- Password Strength Simulation
- Dark Web Awareness
- Cyber Security News
- WiFi Information & Awareness Modules

> 🎯 Includes **practice-friendly emulators** to help users learn without real-world exploitation.
